# Chemistry Primer — Quick Reference for ChemistryInstructor NPC

## Atoms & Elements
- Atoms are the basic building blocks of matter. Each atom has a nucleus (protons + neutrons) and orbiting electrons.
- **Atomic number** = number of protons (defines the element).
- **Mass number** = protons + neutrons.
- **Isotopes**: same element, different number of neutrons.
- **Electron configuration** determines chemical behavior. Valence electrons (outermost shell) participate in bonding.
- The **periodic table** organizes elements by atomic number. Columns = groups (similar properties). Rows = periods.

## Molecules & Compounds
- **Ionic bonds**: electron transfer between metals and nonmetals (e.g., NaCl).
- **Covalent bonds**: electron sharing between nonmetals (e.g., H₂O).
- **Polar covalent**: unequal sharing (water). Nonpolar: equal sharing (O₂, N₂).
- **Hydrogen bonds**: weak attractions between H and electronegative atoms (O, N, F).
- **Molecular formula**: actual atom count (C₆H₁₂O₆). **Empirical formula**: simplest ratio (CH₂O).

## Chemical Reactions
- **Reactants → Products** (mass conserved).
- **Balancing equations**: equal number of each atom on both sides. Use coefficients, never change subscripts.
- **Types**: synthesis (A+B→AB), decomposition (AB→A+B), single replacement (A+BC→AC+B), double replacement (AB+CD→AD+CB), combustion (hydrocarbon+O₂→CO₂+H₂O).
- **Evidence of reaction**: color change, gas bubbles, temperature change, precipitate, odor.

## Stoichiometry
- **Mole** (mol) = 6.022 × 10²³ particles (Avogadro's number).
- **Molar mass** = mass of one mole (g/mol), read from periodic table.
- **Mole conversions**: mass ÷ molar mass = moles; moles × Avogadro's number = particles.
- **Limiting reactant**: the reactant that runs out first, determines product yield.
- **Theoretical yield** = max product from limiting reactant. **Actual yield** = what you get. **Percent yield** = (actual/theoretical) × 100%.

## Acids & Bases
- **Acid**: donates H⁺ (proton donor). Tastes sour, pH < 7.
- **Base**: accepts H⁺ (proton acceptor). Tastes bitter, pH > 7.
- **pH scale**: 0 (strong acid) → 7 (neutral) → 14 (strong base). pH = −log[H⁺].
- **Strong acids**: fully dissociate (HCl, H₂SO₄, HNO₃). **Weak acids**: partially dissociate (acetic acid, carbonic acid).
- **Neutralization**: acid + base → salt + water. HCl + NaOH → NaCl + H₂O.

## Lab Safety Essentials
- Always wear safety goggles, gloves, and a lab coat.
- Know the location of eyewash stations, fire extinguishers, and emergency exits.
- Never taste or directly smell chemicals. Use the wafting technique for odors.
- Dispose of chemicals according to local regulations — never pour down the sink without authorization.
- Report all spills, accidents, and injuries immediately.
